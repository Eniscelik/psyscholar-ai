# Roadmap

## Faz 1 — Proje Hazırlığı

- Ürün dokümanlarının oluşturulması
- MVP kapsamının netleştirilmesi
- Klinik ve psikometrik güvenlik sınırlarının yazılması
- Backend iskeletinin kurulması

## Faz 2 — Akademik Kaynak Arama

- OpenAlex entegrasyonu
- Crossref entegrasyonu
- PubMed entegrasyonu
- Makale metadata normalizasyonu
- DOI eşleştirme

## Faz 3 — AI Cevap Üretimi

- Akademik cevap promptu
- Tez / anti-tez formatı
- Psikometrik test açıklama formatı
- Klinik güvenlik guardrail kontrolü
- Kaynaklı cevap üretimi

## Faz 4 — Frontend MVP

- Ana prompt ekranı
- Kaynak kartları
- Akademik cevap ekranı
- Makale detay ekranı
- Geçmiş aramalar

## Faz 5 — Gelişmiş Özellikler

- PDF yükleme ve makale analizi
- APA 7, BibTeX ve RIS export
- Kullanıcı hesabı
- Türkçe kaynaklar için DergiPark / TR Dizin araştırması
- Vector database ve semantic search

## İlk 30 Günlük Hedef

1. Hafta: Ürün dokümanı ve kapsam
2. Hafta: Backend iskeleti ve API tasarımı
3. Hafta: OpenAlex araması ve makale listesi
4. Hafta: Kaynaklı AI cevap üretimi
